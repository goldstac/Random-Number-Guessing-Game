We welcome all kinds of contributions: fixing bugs, improving the code, adding features, or enhancing the ASCII art.

## How to Contribute

1. **Fork the project** to create your own copy.  
2. **Make your changes** locally.  
   - Keep the code readable and consistent with the existing style.  
   - If adding new ASCII art, try to keep it compatible with the current terminal layout.  
3. **Test your changes**  
   - Make sure the game still runs correctly.  
   - Verify that correct and wrong guesses display the ASCII art properly.  
   - Check hints and random number functionality.  
4. **Submit a Pull Request**  
   - Explain what changes you made and why.  
   - Be friendly and clear.

## Guidelines

- Keep contributions **relevant to the game**.  
- Respect other contributors and maintain a positive, constructive tone.  
- Avoid adding malicious code or unnecessary changes.  

Thank you for helping improve this project!
